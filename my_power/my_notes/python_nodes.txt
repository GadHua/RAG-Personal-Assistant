一、Python 基础语法
1.Python 是解释型、弱类型、跨平台高级编程语言。
2. 特点：语法简洁、代码量少、生态丰富、开源免费。
3. 执行方式：交互式终端、py 文件脚本运行、IDE 运行。
4. 缩进规则：使用缩进划分代码块，代替大括号，缩进必须统一。
5. 注释：# 单行注释，''' ''' / """""" 多行注释。
6. 输出 print ()，输入 input ()，input 默认接收字符串。
二、变量与数据类型
1. 变量：无需声明类型，直接赋值，命名见名知意。
2. 命名规范：字母、数字、下划线，不能数字开头，区分大小写。
3. 禁止关键字命名，如 if、for、def、class。
4. 六大基础数据类型：数字 (int、float、bool、complex)、字符串 str、列表 list、元组 tuple、字典 dict、集合 set。
5. 可变类型：列表、字典、集合。
6. 不可变类型：数字、字符串、元组。
7. 类型转换：int ()、float ()、str ()、list ()、tuple ()。
三、运算符
1. 算术运算：+ - * /// % **。
2./ 浮点除法，// 整数整除，** 幂运算。
3. 赋值运算：= += -= *= /= //=。
4. 比较运算：> < >= <= == !=。
5. 逻辑运算：and 与、or 或、not 非。
6. 成员运算：in、not in，判断元素是否在序列中。
7. 身份运算：is、is not，判断内存地址是否一致。
四、流程控制
1. 条件语句：if、elif、else。
2. 三元表达式：结果 1 if 条件 else 结果 2。
3. 循环结构：for 循环、while 循环。
4.for 常用于遍历序列、容器、range 范围。
5.while 满足条件持续循环，适合未知次数场景。
6. 循环控制：break 终止循环，continue 跳过本次。
7.pass 空语句，占位不执行任何操作。
五、字符串 str
1. 字符串不可变，单引号、双引号、三引号定义。
2. 常用方法：len () 获取长度、strip () 去除首尾空格split () 分割字符串、join () 拼接序列replace () 替换、find () 查找索引、count () 统计次数。
3. 大小写转换：upper ()、lower ()、title ()。
4. 字符串格式化：% 占位、format、f-string5. 切片语法：str [起始：结束：步长]，左闭右开。
六、列表 list
1. 有序、可变、允许重复元素，最常用容器。
2. 定义：[] 中括号，元素可存任意类型。
3. 增：append () 尾部追加、extend () 合并、insert () 指定位置插入。
4. 删：del、pop ()、remove ()、clear ()。
5. 改：通过下标直接赋值修改。
6. 查：下标取值、切片、index ()、count ()。
7. 排序：sort () 原地排序、sorted () 生成新列表。
七、元组 tuple
1. 有序、不可变、元素不可修改。
2. 定义：() 小括号，单个元素末尾需加逗号。
3. 仅可读操作：下标、切片、index、count。
4. 适用场景：保护数据安全、函数多返回值。
八、字典 dict
1. 无序 (3.7 + 有序)、键值对存储、键唯一不可变。
2. 定义：{} 大括号，key:value 格式。
3. 增改：直接赋值、update () 合并字典。
4. 查：[] 取值、get () 安全取值。
5. 删：pop ()、del、clear ()。
6. 常用遍历：keys ()、values ()、items ()。
九、集合 set
1. 无序、元素唯一、自动去重。
2. 定义：{} 或 set ()，空集合只能用 set ()。
3. 添加：add ()、update ()。
4. 删除：remove ()、discard ()、pop ()。
5. 集合运算：交集 &、并集 |、差集 -。
6. 核心用途：数据去重、集合关系判断。
十、函数
1. 函数作用：代码复用、简化结构、模块化。
2. 定义关键字：def 函数名 (参数):。
3. 参数类型：位置参数、关键字参数、默认参数、可变长参数。
4.*args 接收任意位置参数，**kwargs 接收任意关键字参数。
5.return 函数返回值，可返回单个、多个、空。
6. 局部变量：函数内部生效，全局变量：全局生效。
7. 全局关键字 global、nonlocal 嵌套作用域修改变量。
8. 匿名函数 lambda，适合简单逻辑。
十一、容器推导式
1. 列表推导式：[表达式 for 变量 in 序列 条件]。
2. 字典推导式：{k:v for ...}。
3. 集合推导式：{表达式 for ...}。
4. 作用：简化循环代码，一行生成容器。
十二、文件操作
1. 打开文件：open (文件路径，打开模式，编码)。
2. 常用模式：r 读、w 清空写、a 追加写、r + 读写。
3. 读取方法：read ()、readline ()、readlines ()。
4. 写入方法：write ()、writelines ()。
5. 关闭文件：close ()，推荐 with 上下文管理器自动关闭。
6. 编码：一般指定 encoding="utf-8"。
十三、异常处理
1. 异常：程序运行报错，终止代码执行。
2. 结构：try...except...else...finally。
3. 捕获指定异常、多异常捕获、通用异常捕获。
4. 主动抛出异常：raise5.finally 代码无论是否报错都会执行。
十四、模块与包
1. 模块：单个 py 文件，包：多个模块的文件夹。
2. 导入方式：import、from ... import ...。
3. 第三方库安装：pip install 库名。
4. 常用内置模块：os 文件路径、sys 系统参数、time 时间、random 随机数、json 数据处理。
十五、面向对象 OOP
1. 类与对象：类是模板，对象是实例。
2. 定义类：class 类名:，构造方法 init。
3. 属性：实例属性、类属性。
4. 方法：实例方法、类方法 @classmethod、静态方法 @staticmethod。
5. 三大特性：封装、继承、多态。
6. 私有属性：以双下划线__开头，外部无法直接访问。
十六、常用高阶语法
1. 迭代器、生成器 yield，节省内存。
2. 装饰器：@语法，动态扩展函数功能。
3. 闭包：内层函数引用外层变量，延长生命周期。
4. 深浅拷贝：浅拷贝复制引用，深拷贝完全独立数据。
十七、爬虫与基础库
1.json：字典与 json 字符串相互转换。
2.random：生成随机数、随机选择。
3.time/datetime：时间获取、格式化、时间计算。
4.requests：网络请求，爬取网页数据。
5.re 正则表达式：匹配、提取、替换文本。
